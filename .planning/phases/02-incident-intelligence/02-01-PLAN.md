  <name>Task 1: Database Migration — Incident Schema</name>
  <files>backend/internal/database/sqlite.go</files>
  <action>
Add 4 new tables to the `migrate()` function in sqlite.go, AFTER the existing events table creation. Use `CREATE TABLE IF NOT EXISTS` for idempotency. All tables go in a single migration block appended to the existing schema string.

**Table 1: incidents**
```sql
CREATE TABLE IF NOT EXISTS incidents (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    title TEXT NOT NULL,
    description TEXT,
    source_ip TEXT NOT NULL,
    start_time TEXT NOT NULL,
    end_time TEXT NOT NULL,
    event_count INTEGER NOT NULL DEFAULT 0,
    unique_users TEXT,
    unique_ips TEXT,
    unique_hostnames TEXT,
    severity TEXT DEFAULT 'low',
    status TEXT DEFAULT 'new',
    techniques TEXT,
    tactics TEXT,
    mitre_attack_ids TEXT,
    confidence REAL DEFAULT 0.0,
    raw_summary TEXT,
    created_at TEXT DEFAULT (datetime('now')),
    updated_at TEXT DEFAULT (datetime('now'))
);
```

**Table 2: incident_events** (junction table)
```sql
CREATE TABLE IF NOT EXISTS incident_events (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    incident_id INTEGER NOT NULL,
    event_id INTEGER NOT NULL,
    timestamp TEXT NOT NULL,
    source_ip TEXT,
    FOREIGN KEY (incident_id) REFERENCES incidents(id) ON DELETE CASCADE,
    FOREIGN KEY (event_id) REFERENCES events(id) ON DELETE CASCADE,
    UNIQUE(incident_id, event_id)
);
```

**Table 3: techniques** (ATT&CK reference data)
```sql
CREATE TABLE IF NOT EXISTS techniques (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    technique_id TEXT NOT NULL UNIQUE,
    name TEXT NOT NULL,
    description TEXT,
    tactic TEXT,
    url TEXT,
    created_at TEXT DEFAULT (datetime('now'))
);
```

**Table 4: incident_techniques** (junction table)
```sql
CREATE TABLE IF NOT EXISTS incident_techniques (
    incident_id INTEGER NOT NULL,
    technique_id TEXT NOT NULL,
    event_count INTEGER DEFAULT 0,
    PRIMARY KEY (incident_id, technique_id),
    FOREIGN KEY (incident_id) REFERENCES incidents(id) ON DELETE CASCADE,
    FOREIGN KEY (technique_id) REFERENCES techniques(technique_id) ON DELETE CASCADE
);
```

**Indexes** (add after all table creates):
```sql
CREATE INDEX IF NOT EXISTS idx_incidents_source_ip ON incidents(source_ip);
CREATE INDEX IF NOT EXISTS idx_incidents_start_time ON incidents(start_time);
CREATE INDEX IF NOT EXISTS idx_incidents_severity ON incidents(severity);
CREATE INDEX IF NOT EXISTS idx_incidents_status ON incidents(status);
CREATE INDEX IF NOT EXISTS idx_incident_events_incident ON incident_events(incident_id);
CREATE INDEX IF NOT EXISTS idx_incident_events_event ON incident_events(event_id);
```

Implementation notes:
- Append the new SQL to the existing `schema` string variable in `migrate()`
- Do NOT modify the existing events table or its indexes
- Keep the migration idempotent (IF NOT EXISTS on everything)

**Command-line pattern matching** — a slice of `{regex, Technique}` structs for 4 additional techniques not captured by Event ID alone:
- T1003 OS Credential Dumping (credential-access): regex for `mimikatz|kerberos.*ticket|hashdump|lsass`
- T1087 Account Discovery (discovery): regex for `net\s+user|net\s+group|nltest|dsquery`
- T1005 Data from Local System (collection): regex for `copy.*\\\\|xcopy|robocopy.*\/mir`
- T1048 Exfiltration Over Alt Protocol (exfiltration): regex for `curl.*upload|wget.*post|tftp.*put`

**Public functions:**
- `MapEventByEventID(eventID string) []Technique` — lookup in static map
- `MapEventByPatterns(commandLine, processName string) []Technique` — regex matching on combined string
- `MapEventToTechniques(eventID, eventType, commandLine, processName string) []Technique` — combines both, deduplicates by technique ID
- `AllTechniques() []Technique` — returns deduplicated list of all known techniques (for seeding the techniques table)

**mapping_test.go** — Tests covering MITR-01, MITR-02, MITR-03:
- `TestEventIDMapping` — verify each known Event ID maps to correct technique(s)
- `TestPatternMatching` — verify command-line patterns match expected techniques
- `TestMappingCoverage` — count total unique techniques from AllTechniques(), assert >= 10
- `TestDeduplication` — verify MapEventToTechniques returns no duplicate technique IDs
- `TestUnknownEventID` — verify unknown Event ID returns empty slice (no panic)

Use `regexp` stdlib for pattern matching. Compile regex patterns once at package level (var block), not per-call.
