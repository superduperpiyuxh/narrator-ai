Where time actually goes — the hard truths

Prompt engineering eats 3x more time than you budget

Getting the LLM to produce clean, source-traceable narratives without hallucination is not one afternoon. You'll iterate 30–50 prompt versions. Budget 4+ days just for this.

Splunk setup will take longer than expected if you're new to it

Getting a trial instance, ingesting sample security data, and writing SPL queries you've never written before is a 3–5 day learning curve. Don't skip the Splunk docs.

The demo video is a full day's work, minimum

Scripting, recording, editing, narrating, uploading. Most hackers spend 2 hours and it shows. A polished video is literally a judging input — treat it like a deliverable, not an afterthought.

The prompt injection fix is actually fast — don't skip it

Sanitizing user-controlled fields before they enter the LLM prompt takes about half a day. It's the single highest-ROI task for your judge score given the security context of this project.
